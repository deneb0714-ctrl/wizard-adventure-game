## 概要
Pythonとtkinterを用いて制作した2Dアクションゲームです。

プレイヤーは魔法使いを操作し、洞窟内の魔法石を破壊しながらステージを攻略します。
敵モンスターごとに異なる攻撃パターンを実装し、回避アクションとリソース管理を楽しめるゲームにしました。

---

## 使用技術
- Python
- tkinter

---

## 主な機能
- マウス操作によるプレイヤー移動
- 魔法弾による攻撃
- MPシステム
- ステージ制
- 敵cp
- ホーミング弾
- 3way弾
- レーザー攻撃
- 当たり判定
- ゲームオーバー / クリア処理

---

## 工夫した点

### 敵ごとの攻撃パターン実装
ステージごとに敵の行動を変更し、
ホーミング弾・3way弾・レーザーなど複数の攻撃を実装しました。

### リソース管理要素
魔法を無制限に撃てないようMP制を導入し、
時間経過で回復する仕組みにしました。

### シーン管理
「ホーム画面」「ゲーム説明」「ゲーム」「クリア」「ゲームオーバー」など、
ゲームの状態をscene変数で管理しました。

---

## 苦労した点
当たり判定や敵弾管理でバグが発生し、
リスト操作や座標計算を見直して改善しました。
